# PTO – Photography Portfolio Landing Page

## Author
Keiden Wainainai

## Description
This project is a photography portfolio landing page.
It showcases nature, street, and portrait photography.
The website is designed with a simple and clean layout.
It was created as part of a web development assignment.

## Project Setup Instructions
1. Clone the repository from GitHub.
2. Navigate into the project folder.
3. Open the index.html file in a web browser.
